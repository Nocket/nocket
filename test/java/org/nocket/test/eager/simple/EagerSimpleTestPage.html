Test Case: Basic usage of "@Eager" annotation

"@Eager" annotation on the setter method causes, that changes done in the field on the web page are posted immediately to the server. For input fields this occurs while the field lost focus. For check-boxes, select fields etc. it occurs after the change.

Test scenario:

  1. Focus "Text Eager" field. Write input "abc". Leave the field.
  2. Field "Text Eager Echo" equals "abc". Field "Text" remains blank.
  3. Focus "Text" field. Write input "efg". Leave the field.
  4. Field "Text Eager Echo" equals "abc".
  5. Focus "Text Eager" field. Write input "abcd". Leave the field.
  6. Field "Text Eager Echo" equals "abcdef".
