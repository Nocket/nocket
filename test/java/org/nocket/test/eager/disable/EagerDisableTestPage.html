Test Case: Disabling fields by "@Eager" field

Changes done to "@Eager" field should disable / enable depending files. There should be not validation errors shown in such case.

Test scenario:

  1. Click on the "Eager checkbox".
  2. Field "Text" should be disabled.
  3. Click on the "Eager checkbox".
  4. Field "Text" should be enabled.
  5. Press "Save" button.
  6. Field "Text" is marked as invalid now.
