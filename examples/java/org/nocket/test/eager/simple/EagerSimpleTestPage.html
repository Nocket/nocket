Test Case: Basic usage of "@Eager" annotation

Fields: "Text Eager" and "Text Eager Nested" are annotated with "@Eager". Changed done to these fields, are reflected in read-only field - "Text Eager Echo". Field "Text" is not eager. There are no validations.

Test scenario:

  1. Focus "Text Eager" field. Write input "abc". Leave the field.
  2. Field "Text Eager Echo" equals "Text Eager: abc".
  3. Focus "Text Eager Nested" field. Write input "def". Leave the field.
  4. Field "Text Eager Echo" equals "Text Eager Nested: def".
  5. Focus "Text" field. Write input "qwe". Leave the field.
  6. No change.
